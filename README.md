**Steps to Perform**
1. Install the required libraries and tools as stated above
2. Connect the physical device using USB or Start the device on Emulator
3. Start the Appium Server
4. Make sure the apk is on the same directory as the MainApplication.py
5. Run the application using the following command - *python MainApplication.py*
